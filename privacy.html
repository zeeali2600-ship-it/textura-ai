  • Prompt text (only to generate image, not saved permanently).

  • Fix errors, improve stability.

4. Your Control

  • Clear browser storage to reset trials.
  • Stop using app to stop any data transfer.

5. Safety Tips

  • Avoid personal or secret info in prompts.
  • Review image before public use.

1. کیا جمع کیا جاتا ہے

  • پرامپٹ ٹیکسٹ (صرف امیج بنانے کیلئے، مستقل نہیں رکھا جاتا).

  • ٹرائل کاؤنٹر (صرف آپ کے براوزر کے localStorage میں).

3. استعمال

4. آپ کا کنٹرول

  • براوزر اسٹوریج صاف = ٹرائل ری سیٹ.
  • استعمال بند کریں = مزید ڈیٹا نہیں جائے گا.
