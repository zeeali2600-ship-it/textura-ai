Minimal Privacy Policy (Simple Version)

Last Updated:
Support: zerobgmaster.help@hotmail.com

1. What We Collect

  • Prompt text (just to generate the image, not saved).
  • Image URL (temporary, may expire, not stored by us).
  • Trials count (saved only in your browser localStorage).
  • Basic success/error logs (no personal info).

2. What We Do NOT Collect

  • No name, email, phone, address (unless you email support).
  • No payment card data (future billing handled by store/platform).
  • No ads trackers, no analytics profiling.

3. How We Use It

  • Generate images from your prompt.
  • Track free trials locally.
  • Fix errors and keep service stable.

4. Third‑Party

  • Ideogram API (image generation + safety filters).
  • Render (backend hosting, transient request logs).
  • GitHub Pages (static frontend hosting).

5. Your Control

  • Clear browser storage to reset trials.
  • Stop using the app = no further data sent.

6. Safety Tips

  • Avoid personal / secret info in prompts.
  • Generated images may contain artifacts; review before use.

7. Contact

Email: zerobgmaster.help@hotmail.com

8. Consent

By clicking “Generate Image” you agree your prompt goes to the AI service for creation.

1. کیا جمع کیا جاتا ہے

  • پرامپٹ ٹیکسٹ (صرف امیج بنانے کیلئے، محفوظ نہیں کیا جاتا).
  • امیج لنک (عارضی، ہم مستقل نہیں رکھتے).
  • ٹرائل کاؤنٹر (صرف آپ کے براوزر کے localStorage میں).
  • بنیادی کامیابی / خرابی لاگز (کوئی ذاتی معلومات نہیں).

2. کیا جمع نہیں کرتے

  • نام، ای میل، فون، پتہ (جب تک آپ خود ای میل نہ کریں).
  • ادائیگی کارڈ ڈیٹا نہیں.
  • کوئی اشتہاری یا ٹریکنگ اینالیٹکس نہیں.

3. استعمال

  • پرامپٹ سے امیج بنانا.
  • مفت ٹرائل گِننا (لوکل).
  • خرابیاں درست کر کے سروس مستحکم رکھنا.

4. تھرڈ پارٹی

  • Ideogram API (امیج + سیفٹی فلٹر).
  • Render ہوسٹنگ (وقتی لاگز).
  • GitHub Pages (فرنٹ اینڈ اسٹاٹک ہوسٹنگ).

5. آپ کا کنٹرول

  • براوزر اسٹوریج صاف کریں = ٹرائل ری سیٹ.
  • استعمال بند کریں = مزید ڈیٹا نہیں جائے گا.

6. حفاظتی مشورے

  • حساس / ذاتی معلومات پرامپٹ میں نہ لکھیں.
  • امیج میں غلطیاں ہو سکتی ہیں؛ استعمال سے پہلے دیکھ لیں.

7. رابطہ
